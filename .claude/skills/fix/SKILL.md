# Fix an Error

Problem: $ARGUMENTS

1. Read the error message / traceback carefully
2. Identify the root cause (not symptoms)
3. Find the relevant code
4. Apply the minimal fix
5. Re-run to verify the fix works
6. If the fix introduces a new error, repeat

Do not add try/except blocks or error suppression as a fix. Fix the actual problem.
