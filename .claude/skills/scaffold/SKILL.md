# Scaffold a New Project

Set up a project for $ARGUMENTS.

1. Create a minimal project structure appropriate for the task
2. Initialize with the right package manager / dependency tool
3. Install only essential dependencies
4. Create a basic entry point that runs and shows something
5. Start the dev server or run the entry point to verify it works
6. Report what's running and where (URL, port, etc.)

Keep it minimal — only add what's needed to start building.
